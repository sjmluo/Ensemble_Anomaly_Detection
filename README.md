## Requirements
- Python >= 3.6

## Installation
To get started, run
```
$ git clone https://github.com/sjmluo/Ensemble_Anomaly_Detection.git
$ cd Ensemble_Anomaly_Detection
$ pip install -r requirements.txt
```

To download the datasets, run
```
$ ./get_data.sh
```
